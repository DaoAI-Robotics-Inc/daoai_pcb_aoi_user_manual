焊料检测 2D（Solder 2D，基于颜色比例）
==========================================

**此页面的用途**

在焊料 ROI 内用 HSV 颜色范围对像素二值化，计算有效比例并与设定范围比较以判定 OK/NG，适用于 SMD 焊点的焊料检测。

**如何进入**

模板编辑器中绘制对应 ROI 后，在参数面板中配置该工具的参数。

**操作流程**

**核心思路**：在焊料 ROI 内用 **HSV 颜色范围** 对像素二值化，计算 **有效比例** 并与设定范围比较以判定 OK/NG。一个 ROI 可配置多组颜色范围（如蓝通道斜面焊料、红通道平面焊盘），分别独立设阈值。

      :scale: 50%
      :alt: 焊料2D ROI

**参数说明**

- **颜色公式（Color Formula）**：选择 HSV 或三色（Tricolor）公式；选择三色时配置 **三色 X / Y / Z / A** 系数。
- **中心色调（Center Hue）/ 中心饱和度（Center Saturation）**：HSV 模式下色相基准点（位于色盘内部）。
- **起始色调（Start Hue）/ 终止色调（End Hue）**：色盘圆环边缘的两点，与中心点共同构成扇形有效色相范围。
- **起始明度（Start Value）/ 终止明度（End Value）**：限定有效像素的明度区间。
- **有效比例范围（Valid Ratio Ranges）**：每组颜色范围对应的有效比例上下限；落入范围内判 OK。
- **有效比例列表（Valid Ratio List）**：评估样本得到的各 ROI 有效比例统计，辅助设定合理范围。
- **启用可视化（Enable Visualization）**：显示二值化 Mask 与比例统计（仅建议调试时开启）。
- **使用共享参数（Use Shared Parameters）**：开启后，对同一组件下所有同类型特征共用一套参数；关闭后可对单个特征单独调参。

**三角形抽色（HSV 三角法）**

颜色范围支持 **三角形抽色** 配置：在 HSV 颜色空间中，用一个三角形圈出目标颜色簇。三角形由三个顶点定义，分别对应中心色、起始色与结束色，比传统的"矩形" HSV 范围更紧凑、误抽其他颜色更少。可点击 HSV 颜色块拖动顶点；也可在画布上 **按住 Ctrl 并点击像素** 直接采集目标颜色作为三角形顶点。

**多范围间的逻辑关系**

可同时配置多个颜色范围（例如：蓝色焊点 + 红色助焊剂）。系统对每个范围分别做二值化统计，最终判定时 **取所有范围的并集**：

* 任一颜色范围匹配的像素都会被纳入"焊锡有效像素"统计。
* 颜色范围之间是 **OR** 关系，不会因为某一范围阈值过严而漏判其他颜色覆盖区域。
* 各范围的统计比例可在右侧参数面板中独立查看。

如果不需要某个范围，可将其 *阈值* 调至 0，或在颜色范围列表中删除该范围。

**画布快捷键 — 提取颜色**

* **Ctrl + 鼠标左键**：在画布上单击像素，把该像素颜色作为新的三角形抽色顶点。
* **Ctrl + 拖动**：框选一块小区域，把区域内的平均颜色作为顶点。
* 切到 **抽色二值化预览** 视图（参数面板顶部 toggle），可实时查看当前颜色范围抽到的像素分布，便于调参。


**常见问题与提示**

.. tip::

   - **焊锡阈值参数被 AI 更新拉飞**：避免把分数远超阈值的焊锡反馈为"无错误"——这会让 AI 学到错误的阈值。可在 **系统设置 → 模型更新器** 关闭 **AI 更新覆盖**，关闭后 AI 更新最多调整到设定的比例上下限，不会大幅突破。
   - **参数更新计算不准确**：检查训练集中是否有打分极端的反馈图片，移除明显错误或极端的反馈后重新训练。也可关闭 **更新颜色范围 / 更新颜色比例**，关闭后编程页面"参数更新"按钮将不会改动对应参数。
