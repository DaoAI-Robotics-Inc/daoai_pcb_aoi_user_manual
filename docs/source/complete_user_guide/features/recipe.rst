产品设置（Recipe）
=====================

产品设置区（**设置**，即 Recipe）位于编程页面（Teach）左侧标签栏的顶部区域。该区域包含五个子标签，用于配置产品的传送带、PCB 物理参数及检测判废逻辑。子标签依次为：

- **PCB详情** （PCB Detail）
- **传送带设置** （Conveyor Setup）
- **PCB尺寸** （PCB Dimension）
- **全PCB拍摄** （Full PCB Capture）
- **检测设置** （Inspection Settings）

每个子标签完成后会显示蓝色对勾图标，表示该步骤已配置完成。后四个子标签在完成前置步骤之前处于禁用状态，须按顺序操作。

**PCB详情（PCB Detail）**

PCB详情标签页包含两个区域：

- **产品名称（产品名称）**：可在此输入或修改产品名称。产品名称须唯一；若与已有产品重名，保存时会提示错误。点击 **保存** 提交修改，点击 **取消** 放弃编辑。

- **检测参数（Specs for Inspection）**：以列表形式显示四项产品参数的当前状态，每项均有操作按钮可跳转至对应子标签页：

  - **传送带宽度（Conveyor Width）**：未设置时显示 ``(未定义)``，已设置时显示毫米数值。点击 **设置** 跳转至 **传送带设置** 子标签。
  - **PCB尺寸（PCB Dimension）**：未测量时显示 ``(未定义)``，已测量时显示宽 × 高（mm）。点击 **测量** 跳转至 **PCB尺寸** 子标签。
  - **全PCB拍摄（Full PCB Capture）**：未拍摄时显示 ``(未定义)``，已拍摄时显示已采集状态。点击 **拍摄** 跳转至 **全PCB拍摄** 子标签。
  - **检测设置（Inspection Settings）**：显示当前判废模式摘要（见下方说明）。点击 **配置** 跳转至 **检测设置** 子标签。

**传送带设置（Conveyor Setup）**

传送带设置用于在编程过程中锁定传送带并调整宽度，以便手动放板或使用 PCB 进入功能。

操作步骤：

1. **选择一个传送带（Select a Conveyor）**：从下拉列表中选择当前可用的传送带槽位。
2. **锁定传送带（Lock Conveyor for this Task）**：点击此按钮锁定所选传送带供本次编程使用。
3. **设置传送带宽度（Set Conveyor Width）**：在输入框中输入目标宽度（mm），再点击 **移动到宽度（Move to Width）** 使传送带实际移动到该宽度。当前传送带宽度会显示在下方以供参考。
4. **保存并继续（Save and Continue）**：确认宽度后点击此按钮，将宽度写入产品并跳转至 **PCB尺寸** 子标签。需已完成锁定方可启用。
5. **释放传送带（Release Conveyor）**：操作完成后点击此按钮，释放对传送带的独占控制权。

**PCB尺寸（PCB Dimension）**

该标签页用于测量或输入 PCB 的物理尺寸（宽度与高度，单位 mm）。操作流程与创建产品向导中的 **输入或计算尺寸** 步骤一致，详见 :ref:`创建产品` 中的对应说明。

**全PCB拍摄（Full PCB Capture）**

该标签页用于重新执行完整 PCB 拍摄，生成用于检测的黄金图像。操作流程与创建产品向导中的 **完整 PCB 拍摄** 步骤一致，详见 :ref:`创建产品` 中的对应说明。完成拍摄后可在此处更新相机参数（曝光、增益、白平衡等）并重新拍摄覆盖当前版本。

**检测设置（Inspection Settings）**

检测设置标签页用于配置拼板产品中 **子板（sub-PCB）** 的自动判废逻辑，决定系统如何判断每块子板为好板或坏板。

可从 PCB详情 标签页点击 **检测设置** 行右侧的 **配置** 按钮进入此标签页。

该标签页包含以下四项参数：

**使用NG元件比例（Use NG component ratio）**

Switch 开关，默认 **开启**。启用后，系统根据子板上 NG 元件的比例来自动判废。开启时，下方显示以下子项：

- **失败比例阈值（Failure ratio threshold）**：InputNumber 输入框，有效范围 ``0.0–1.0``，默认值 ``0.5``。当子板的 NG 元件比例超过该阈值时，子板被判定为坏板。

**使用坏板标记元件（Use badmark elements）**

Switch 开关，默认 **关闭**。启用后，系统依据产品中已定义的坏板标记元件（badmark）来判废子板（参见 :ref:`检测控制与特殊标记`）。开启时，下方显示以下子项：

- **组合逻辑（Combine logic）**：Radio 单选，可选：

  - **任一NG**：只要有任意一个坏板标记元件判为 NG，子板即判为坏板。
  - **全部NG**：所有坏板标记元件均判为 NG，子板才判为坏板。

- **已定义坏板标记元件数（Badmark elements defined）**：显示当前产品中已定义的坏板标记元件数量（只读）。

- 若启用了坏板标记模式但尚未定义任何坏板标记元件，页面会显示黄色警告提示，说明该模式不会生效。

**两种模式均关闭时**

若 **使用NG元件比例** 和 **使用坏板标记元件** 均关闭，页面会显示提示信息：子板不会被自动判定，仅进行人工复检。

**保存与取消**

编辑完成后点击 **保存** 提交修改；点击 **取消** 恢复为上次已保存的值。提交前系统会校验 **失败比例阈值** 是否在 ``[0, 1]`` 范围内，超出范围将提示错误并阻止保存。

**PCB详情中的检测设置状态摘要**

PCB详情标签页的 **检测设置** 行会根据当前配置显示以下摘要文字：

- **比例 · 坏板标记**：两种模式均已启用。
- **仅比例**：仅启用了NG元件比例模式。
- **仅坏板标记**：仅启用了坏板标记模式。
- **仅人工复检**：两种模式均已关闭。
