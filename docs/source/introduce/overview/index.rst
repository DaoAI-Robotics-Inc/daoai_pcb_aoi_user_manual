系统概览与界面导航
==============================

本页介绍 AOI 系统的整体架构、顶层界面导航逻辑，以及首次启动时的登录操作与启动画面。面向所有角色（管理员、操作员、编程人员）。

   :local:
   :depth: 1

访问系统
------------------------------

浏览器直接访问系统根路径（默认 ``http://localhost:3005/``），或在桌面点击启动器后浏览器自动打开。未登录时会自动跳转至登录页。

.. image:: ../quick_start/images/login.png
   :scale: 60%
   :alt: AOI 系统登录页

系统整体布局
------------------------------

登录后，所有页面共享一个固定的顶部导航栏。导航栏包含以下元素：

- **左上角 Logo / 系统名称**：点击可随时从任何页面返回 :ui-tab:`主页`。在 :ui-tab:`设置` 和 :ui-btn:`工作列表` 页面，名称旁会额外显示当前页面的标签；在 :ui-tab:`产品编程页面`，会在名称旁追加正在编程的产品名称。
- **右上角功能区**：从左到右依次为磁盘用量指示器、用于进入 :ui-tab:`设置` 的 :ui-btn:`齿轮图标`，以及当前登录的用户名。

.. note::

   检测会话运行期间，点击 :ui-btn:`齿轮图标` 会弹出提示，告知设置暂不可用；需先停止检测任务后再进入设置。

登录页面
------------------------------

.. image:: ../quick_start/images/login.png
   :scale: 60%
   :alt: 登录页面

登录页包含以下控件：

- :ui-param:`用户名`：输入账户名称。
- :ui-param:`密码`：输入账户密码，支持按回车键直接提交。
- :ui-btn:`登录`：提交凭据，验证通过后跳转至 :ui-tab:`主页`。
- :ui-btn:`更改主机地址`：打开主机地址配置弹窗，用于修改后端服务器地址（默认 ``http://localhost:8000/``）。如需从其他设备访问，可在此将 ``localhost`` 替换为目标机器的 IP 地址。

界面语言可在登录后于 :ui-tab:`系统设置` → :ui-tab:`语言` 中切换（:ui-opt:`中文` / :ui-opt:`英文`）。

.. image:: ../quick_start/images/host_addr.png
   :scale: 60%
   :alt: 更改主机地址弹窗

用户角色
------------------------------

系统内置三种用户角色，每种角色对应不同的操作权限：

- **管理员（ADMIN）**：拥有全部权限，包括 :ui-tab:`团队` 管理（创建/删除账户、分配角色）、所有系统设置项，以及主页上的全部功能区（包括产品编程入口 :ui-btn:`训练PCBA` 和 :ui-btn:`工作列表`）。
- **编程人员（PROGRAMMER）**：可使用产品编程（:ui-btn:`训练PCBA`）和检测相关功能，并访问大多数系统设置，但无法访问 :ui-tab:`团队` 管理页面。
- **操作员（OPERATOR）**：仅可执行检测操作（启动/停止检测任务、查看 :ui-btn:`工作列表`），不可访问产品编程入口，且系统设置中多数高级选项对操作员不可用。

管理员账户在系统部署时提供，默认用户名为 **admin**。账户管理请进入设置页的 :ui-tab:`团队` 分组（仅管理员可见）。

系统初始化页面
------------------------------

系统首次启动或后端服务尚未就绪时，浏览器会进入 **系统初始化中…** 等待页面。页面显示初始化进度或连接状态，并提供两个操作按钮：

- :ui-btn:`刷新`：重新加载页面，重新尝试连接后端。
- :ui-btn:`更改主机地址`：在后端地址填写有误时，可打开主机地址弹窗重新配置。

当后端初始化完成后，系统将自动根据登录状态跳转：令牌有效则直接进入主页，令牌已过期则跳转至登录页。

注意事项
------------------------------

.. note::

   - 检测会话进行中，点击顶部导航栏的 Logo 返回主页、或点击 :ui-btn:`齿轮图标` 进入设置时，若当前正在复检页面且有未完成的确认操作，系统会先弹出确认弹窗，待操作员确认后再执行导航，以防止意外丢失检测数据。
   - 角色权限在登录时由服务器签发的 JWT 令牌确定，切换权限需重新以对应账户登录。
